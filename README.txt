# What is it?
A 2-player game with a field consisting of 9 squares.
The players choose one after the other a field in which they want to put their mark/token. 
The goal is to place 3 of their own tokens in a row, column or diagonal.

# How to run?
- open soluton in visual studio
- Set Release, x64
- Build -> build solution
- Sobald das fertig ist: 
- via the console: navigate to the executable file (\code_examples\TicTacToe\x64\Release)
  and run TicTacToe.exe 

- via visual Studio: out of the TicTacToe folder open TicTacToe.sln
  and press run.

# Implemented with VisualStudio version:
Microsoft Visual Studio 2019
